# Cloudflare Integration

## Purpose

Cloudflare is the DNS and domain control plane for AJ Digital public and app domains.

## Expected Uses

- Manage DNS for `audiojones.com` and `ajdigital.app`.
- Support wildcard subdomains for product and tenant routing.
- Coordinate proxy, certificate, and edge settings.
- Provide future storage or edge services if approved.

## Approval Rule

DNS, wildcard, certificate, firewall, redirect, and production routing changes require explicit human approval.
