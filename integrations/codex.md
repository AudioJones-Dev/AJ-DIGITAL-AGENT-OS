# Codex Integration

## Purpose

Codex Desktop is used for local repository inspection, scoped edits, validation, and Git-aware execution.

## Expected Uses

- Workspace audits.
- Documentation creation.
- Code edits within explicit scope.
- Local command validation.
- Git status inspection.

## Boundaries

Codex must respect task-specific path boundaries, avoid secrets, preserve active work, and require approval before commits, pushes, remotes, installs, migrations, deployments, or destructive actions.
