# Claude Code Integration

## Purpose

Claude Code is a planned repo-aware coding assistant lane for future AJ Digital implementation work.

## Expected Uses

- Implement scoped code changes.
- Assist with refactors where assigned.
- Work from approved specs and narrow prompts.
- Coordinate with Codex and Claude CoWork through clear handoffs.

## Boundary

Claude Code should not receive broad overlapping prompts that duplicate Codex execution scope. Each task should identify the repo, allowed files, blocked files, validation requirements, and approval gates.
