# Obsidian Integration

## Purpose

Obsidian supports human-facing knowledge management, strategic context, and control-room workflows.

## Expected Uses

- Capture durable decisions.
- Maintain briefs and planning notes.
- Prepare Claude and Codex prompts.
- Store non-secret operational references.

## Boundary

Agent OS repository files remain the source of truth for registry, policies, schemas, and deployment topology. Obsidian can reference and explain those files but should not silently replace them.
