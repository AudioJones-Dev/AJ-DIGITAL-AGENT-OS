# Claude CoWork Policy

## Purpose

Claude CoWork is a planning, synthesis, documentation, and strategy lane. It should complement Codex without duplicating execution scope.

## Appropriate Tasks

- Draft product and architecture briefs.
- Review specs and identify gaps.
- Maintain control-room or vault documentation.
- Produce decision options and operator prompts.
- Summarize repo findings provided by Codex.

## Boundary

Claude CoWork should not be assigned the same broad execution prompt as Codex. Codex owns local repo inspection, code edits, validation, and Git-aware execution when those actions are approved.

## Handoff Rule

Claude outputs should be converted into scoped Codex tasks before implementation begins.
