# Hermes Control Plane

## Status

Planned

## Product URL

```txt
hermes.ajdigital.app
```

## Purpose

Hermes Control Plane is planned as the orchestration, memory, routing, and policy enforcement layer for AJ Digital agents.

## Initial Responsibilities

- Route tasks across approved tools and agent lanes.
- Maintain agent context and memory boundaries.
- Enforce human approval gates.
- Provide audit logs for agent actions and decisions.
- Coordinate with Codex, Claude, OpenClaw, GitHub, Cloudflare, and Vercel where approved.

## Approval Notes

Hermes must default to read-only inspection until explicit policy grants a narrower action scope.
