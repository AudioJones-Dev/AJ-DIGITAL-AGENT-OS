# Diagnostics Agent

## Status

Planned

## Product URL

```txt
diagnostics.ajdigital.app
```

## Purpose

The Diagnostics Agent is planned as a bug, project, and production-readiness diagnostic assistant for AJ Digital workspaces.

## Initial Responsibilities

- Inspect project state and report blockers.
- Identify missing configuration, failing checks, and deployment gaps.
- Produce operator-ready repair plans and validation summaries.
- Preserve active work and avoid destructive actions without approval.

## Approval Notes

The Diagnostics Agent may inspect repositories according to task scope. It must not modify product code, secrets, deployment settings, or remotes unless explicitly authorized.
