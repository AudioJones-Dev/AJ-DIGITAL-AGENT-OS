# Vercel Deployment Model

## Purpose

Vercel is the default planned hosting platform for AJ Digital web applications and agent dashboards unless a product-specific spec chooses another target.

## Expected Uses

- Public product apps.
- Agent dashboards.
- API routes and serverless functions.
- Preview deployments for review.
- Production deployment after approval.

## Required Checks

- Confirm project link and team before deployment.
- Confirm environment variables without printing values.
- Run approved validation commands.
- Verify preview or production route after deployment.
- Record deployment URL and rollback options.

## Approval Rule

Deployments, promotions, project setting changes, and environment changes require explicit human approval.
