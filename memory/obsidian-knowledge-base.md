# Obsidian Knowledge Base

## Purpose

Obsidian is planned as a human-readable knowledge base for AJ Digital operating context, briefs, decisions, and long-term notes.

## Use Cases

- Store architecture decisions.
- Maintain control-room notes.
- Capture product and client context.
- Prepare implementation prompts.
- Preserve runbooks and incident records.

## Boundary

Obsidian notes can inform Agent OS but should not replace source-of-truth files in this repository for registry, policy, schema, or deployment topology.
